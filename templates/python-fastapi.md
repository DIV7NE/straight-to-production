## Code Standards
### Always Do
1. Pydantic models for ALL request/response validation — never trust raw input
2. Business logic in `services/`, not route handlers
3. `Depends()` for DB sessions, auth, and shared logic
4. Async `sessionmaker` with `async with` for every DB operation
5. Specific HTTP status codes — 201 created, 204 deleted, 422 validation
6. Config via `pydantic-settings` from environment variables

### Never Do
1. Never `db.commit()` outside a session context manager
2. Never return ORM models directly — serialize through Pydantic schemas
3. Never catch bare `Exception` — catch specific errors, let 500s propagate
4. Never use `*` imports — breaks type checking, creates hidden coupling
5. Never skip `max_length` on string fields — unbounded input is a DoS vector
6. Never use sync I/O in async routes — blocks the event loop

## Stack Patterns
### Auth Middleware
```python
# app/dependencies.py
from fastapi import Depends, HTTPException, status
from fastapi.security import HTTPBearer, HTTPAuthorizationCredentials
from jose import JWTError, jwt

security = HTTPBearer()
async def get_current_user(
    creds: HTTPAuthorizationCredentials = Depends(security),
    db: AsyncSession = Depends(get_db),
) -> User:
    try:
        payload = jwt.decode(creds.credentials, settings.SECRET_KEY, algorithms=["HS256"])
        if (user_id := payload.get("sub")) is None:
            raise HTTPException(status_code=status.HTTP_401_UNAUTHORIZED)
    except JWTError:
        raise HTTPException(status_code=status.HTTP_401_UNAUTHORIZED)
    user = await db.get(User, user_id)
    if not user:
        raise HTTPException(status_code=status.HTTP_401_UNAUTHORIZED)
    return user
```
### Input Validation
```python
# app/schemas/post.py
from pydantic import BaseModel, Field, ConfigDict
class PostCreate(BaseModel):
    title: str = Field(..., min_length=1, max_length=200, strip_whitespace=True)
    content: str = Field(..., min_length=1, max_length=50000, strip_whitespace=True)
    is_published: bool = False
class PostResponse(BaseModel):
    model_config = ConfigDict(from_attributes=True)
    id: int; title: str; content: str; author_id: int; created_at: datetime
```
### Error Handling
```python
@router.post("/", response_model=PostResponse, status_code=status.HTTP_201_CREATED)
async def create_post(body: PostCreate, user: User = Depends(get_current_user),
                      db: AsyncSession = Depends(get_db)):
    try:
        return await post_service.create(db, user_id=user.id, data=body)
    except IntegrityError:
        raise HTTPException(status_code=409, detail="Post with this title already exists")
```
